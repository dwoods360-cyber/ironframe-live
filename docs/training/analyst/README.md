# Jr. GRC Analyst track — Ironframe Core practicum

**Audience:** Students who finished Ironframe Core (student track)  
**Reading level:** 11th grade  
**Goal:** Run a supervised junior GRC workflow inside Ironframe Core

This track is **Path A** after [student Core](../student/README.md). It does **not** cover Ops Hub or GTM sales queues.

---

## Start here

1. Confirm student Core certification is done  
2. [ANALYST-INDEX](./ANALYST-INDEX.md) — **knowledge gate first** (manuals + 40-question exam ≥ 80%), then labs  
3. Keep a journal: route · finding · evidence · GRC terms · time  

**GRC test readiness:** Manuals cover CIA, control types, SLE/ARO/ALE, SOC 2 / ISO 27001 / NIST CSF **recognition** (industry literacy — Ironframe is not a SOC company), evidence & audit language — then Ironframe labs prove you can operate the platform.
