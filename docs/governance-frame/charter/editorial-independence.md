# Governance Frame Editorial Independence

Authoritative detail: [`editorial-standards.md`](./editorial-standards.md) §1.

Summary:

- Governance Frame is not an Ironframe marketing site.
- Ironframe should almost never be the subject of the research.
- Conclusions must not be written backward from product architecture.
- Ironframe may appear only as a clearly disclosed implementation example.
- Never imply that a regulator, law, or standard requires Ironframe.
- “Independent” means editorially independent.
