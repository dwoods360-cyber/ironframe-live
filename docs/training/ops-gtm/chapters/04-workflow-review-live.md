## Why this chapter matters

You host the 15-minute diligence call. LIVE assist is a **sidecar** (mic STT, Pocket Q&A, recap) — agents do not run the meeting.

## Learning objectives

- Open the LIVE page and set Company / Buyer fields.
- Start mic LIVE long enough to see level activity or words (lab safe room).
- End LIVE → confirm a recap card can appear (or log the blocker).
