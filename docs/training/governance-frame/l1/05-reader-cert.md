# L1 Ch.5 — Reader quiz + capstone (GFP-Reader)

> **Track:** GF_L1 · **Lab:** GF-R-CERT

## Requirements

1. [Reader quiz](../manuals/l1-reader-quiz.md) ≥ 80% (16/20).  
2. **Capstone:** one annotated briefing packet:
   - Link/slug of briefing  
   - ≥ 5 taxonomy labels  
   - ≥ 1 primary-source trace (Ch.4)  
   - One paragraph: “What this briefing is **not** claiming”  
3. Trainer sign-off → **GFP-Reader** badge ([GFP-CERTIFICATION](../GFP-CERTIFICATION.md)).
