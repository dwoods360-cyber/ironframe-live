# GFP-Verifier portfolio rubric

**Pass:** all required rows **Meets** or better + completed promote/deny sheet.

| # | Criterion | Required | Evidence |
|---|-----------|----------|----------|
| 1 | Source inspection notes (opened/checked authorities) | Yes | Review log |
| 2 | Verification statuses assigned (e.g. Verified / Qualified / Unsupported) | Yes | Ledger |
| 3 | Independence / COI check documented | Yes | Checklist |
| 4 | Corrections triggers identified (if any) | Yes | Notes |
| 5 | Desk findings summarized without rubber-stamping “ready” | Yes | Review memo |
| 6 | Promote **or** Deny recommendation with rationale | Yes | Decision sheet |
| 7 | Explicit: human Publisher still owns production Approve | Yes | Attestation line |
| 8 | No unsupervised cold publish by learner | Yes | Trainer confirm |
