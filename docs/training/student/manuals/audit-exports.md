# Audit exports — Ironframe Core (student)

**Track:** Student · **Reading level:** 11th grade · **Live route:** `/exports`

Download a CSV or PDF for the **active tenant**. Use it when a lab asks for auditor-style proof.

> Old links to `/dashboard/exports` send you to `/exports`. Always use **`/exports`**.

---

## Before you download

Your classroom tenant may unlock exports only when setup gates pass. If a banner blocks download:

1. Read the banner text.
2. Ask your trainer whether Get Started / billing gates apply in class.
3. Do not bypass gates with admin tools unless your trainer says so.

---

## Walkthrough

1. Confirm the active tenant name in the UI (classroom seed or assigned slug).
2. Open **`/exports`**.
3. Choose **CSV** or **PDF**.
4. Save the file with a timestamp and tenant name in your lab journal.

Do **not** use PILOT vendor screens for auditor CSV. Those pages show demo data only. See [pilot vs preview](./pilot-vs-preview.md).
