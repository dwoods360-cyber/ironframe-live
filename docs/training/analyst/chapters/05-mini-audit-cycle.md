# Chapter 5 — Mini audit cycle (end-to-end)

> **Track:** ANALYST · **Reading level:** 11th grade · **Release:** `v0.1.0-ga-epic17`  
> **Primary route:** `/integrity` · **Lab IDs:** ANALYST-005

## Why this chapter matters

Real junior work is a **cycle**, not four separate clicks. You run hazard → evidence → export → board note in one timed pass.

## Learning objectives

When you finish, you can:

- Complete one full cycle in under 30 minutes (or trainer timebox).
- Point to screenshots or journal lines for each step.
- List one gap (missing evidence, blocked export, unclear hazard).

## How to get there

1. Start at `/integrity` with a fresh journal page titled “Mini cycle.”
2. Re-run Chapters 1–4 quickly on **one** hazard (may be the same as before).
3. Stop when the board note is done or the timebox ends.
