# Path D — Governance Frame training

**Audience:** Learners who write, read, or assure governance research (not Ironframe sales)  
**Reading level:** 11th–12th grade plain English  
**Classroom:** Curriculum lives under this folder (`/docs/training/governance-frame` after seed)  
**Discovery:** Public research chrome — `https://research.ironframegrc.com/training` or local `/gf-research/training`  
**Credential:** **Governance Frame Practitioner (GFP)** — stackable L1 → L2 → L3
