# Jr. GRC Analyst Index — Core practicum

**Track:** ANALYST · **Reading level:** 11th grade · **Release:** `v0.1.0-ga-epic17`  
**Prerequisite:** [LEVEL1-STUDENT-INDEX](../LEVEL1-STUDENT-INDEX.md) (Ironframe Core) complete

Work the **knowledge gate** first, then lab chapters. Labs alone are not enough to pass a GRC knowledge test — manuals + practice exam are required.

**Certification bar:** (1) practice exam **≥ 80%** · (2) Chapters 1–5 labs Pass · (3) trainer sign-off on Ch.6 packet.

1. Classroom host and credentials from your trainer.

3. Open the [UI lab checklist](../student/manuals/ui-lab-checklist.md) if navigation is rusty.
