## Beachhead segments (score 1–5 each)

| Segment | Buying trigger | Ironframe wedge | Disqualify if… |
|---------|----------------|-----------------|----------------|
| **Multi-entity holding / regional BHC** | Affiliate audit walls; board asks for $ risk | Tenant isolation + BigInt ALE | Only needs SOC 2 speed + 400 connectors |
| **Public power / mid-market utility** | NERC CIP + OT/IT evidence burden | Ops GRC + physical-unit sustainability path | Pure retail “compliance software” RFP |
| **MSSP / vCISO (5–50 clients)** | Client evidence cross-bleed fear | Hard enclave per client; export as deliverable | Wants white-label checkbox marketplace |
| **Regulated mid-market (health / fintech seed)** | First GRC hire; questionnaire pain | Path B $4,999 co-builder → planned ~$35k GA | Unwilling to pay a pilot |

**Illustrative public profiles (research only):** Zions Bancorporation (multi-affiliate), Customers Bancorp–class regionals, SMUD / Salt River Project–class utilities, Apptega/GetCybr-category MSSP buyers. Never use engineering demo slugs (`medshield`, `vaultbank`, `gridcore`) as prospect names.

## ICP scorecard (min 12/20 to shortlist)

| Dimension | 1 | 3 | 5 |
|-----------|---|---|---|
| Regulated pressure | Low | Active audit / questionnaire | Board / regulator clock |
| Multi-entity or multi-client | Single org sheet | 2+ entities | Strict isolation needed |
| Quantification appetite | Heatmaps OK | Mixed | Dollars required |
| Budget / willingness to pay | Free only | Pilot OK | $4,999+ Path B |
| Access to economic buyer | Gatekeepers only | Champion | CISO / VP / partner |

## Trigger signals (Ironleads / Scout)

- Funding round + security hire  
- First dedicated GRC / compliance role  
- Enterprise vendor questionnaire surge  
- Auditor engagement kickoff  
- Public fine / consent order in sector (ethically cited as sector pressure, not shaming)
