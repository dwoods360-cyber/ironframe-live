# Governance Frame Practitioner (GFP) — certification

**Credential name:** Governance Frame Practitioner  
**Stack:** GFP-Reader → GFP-Writer → GFP-Verifier  
**Authority corpus:** GF-STANDARDS-001 (`docs/governance-frame/charter/editorial-standards.md`)

---

## What GFP certifies

| Skill | Industry value |
|-------|----------------|
| Separate fact / interpretation / recommendation | Board-ready writing |
| Primary-source citation discipline | Audit & policy credibility |
| Regulatory precision (law vs guidance) | Compliance analyst work |
| No invented statistics / vendor spin | Trust with executives |
| Quarantine vs publish judgment | Editorial assurance |

## What GFP does **not** certify

- ISO 27001 Lead Auditor / Implementer  
- ISACA CISA / CRISC / CISM  
- CompTIA Security+  
- **SOC 2 Type I/II** (or any AICPA attestation) — GFP is not a SOC credential; Ironframe is not a SOC audit firm  
- Authority to speak as counsel or a regulator  
- Unsupervised production publish rights (Publisher still Approves)

SOC / ISO / NIST **recognition** for job interviews lives in the [Analyst](../analyst/ANALYST-INDEX.md) and [Practitioner Core](../practitioner-core/PRACTITIONER-CORE-INDEX.md) tracks — not in GFP exams.
