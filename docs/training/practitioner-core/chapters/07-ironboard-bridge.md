# Chapter 7 — IronBoard telemetry bridge

> **Track:** PRACTITIONER_CORE · **Reading level:** 11th–12th grade · **Release:** `v0.1.0-ga-epic17`  
> **Primary route:** `/cockpit` · **Lab IDs:** PCORE-007

## Why this chapter matters

The Workforce Cockpit shows agent/workforce activity that can bridge to advisory board tools. You learn to **observe** telemetry, not to treat chat as a source of truth for ALE.

## Learning objectives

- Open `/cockpit` and describe one agent or activity signal.
- State that board chat does not replace Integrity Hub dollars.
- Note when IronBoard access is trainer-gated.
