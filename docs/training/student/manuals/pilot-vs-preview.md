# Pilot vs preview badges (student)

**Track:** Student · **Reading level:** 11th grade

Some menu links show a badge. The badge tells you whether the page is live Core work, demo data, or unfinished.

---

## Badge meanings

| Badge | What it means in class |
|-------|-------------------------|
| **PILOT** | Demo / seed data only. Not a live company database. |
| **PREVIEW** | Early or incomplete module. Some roles cannot open it. |
| **STAGED DRAFT** | Internal staging label. Rare in student nav. |

---

## Surfaces you may see

| Route | Badge | Notes |
|-------|-------|-------|
| `/vendors` | PILOT | Seed vendor demo data |
| `/vendors/supply-chain` | PILOT | Same seed boundary |

PILOT pages often show: **Pilot surface — seed data only**.

For auditor-style files in Core labs, use **`/exports`**, not PILOT vendor CSVs.
