# Chapter 5 — Calendar & recap push

> **Track:** OPS_GTM · **Reading level:** 11th–12th grade · **Release:** `v0.1.0-ga-epic17`  
> **Primary route:** `/dashboard/operations?tab=calendar` · **Lab IDs:** OPSGTM-005

## Why this chapter matters

After a call, action items should land on the Ops Calendar so follow-up is visible.

## Learning objectives

- Generate or open a call recap on the LIVE page.
- Use **Push to calendar** when available.
- Find the resulting card on the Calendar tab (or log the error).

## Lab — Push once (OPSGTM-005)

1. From a lab recap, click Push to calendar (or trainer runs the smoke script).
2. Open `/dashboard/operations?tab=calendar`.
3. Journal card title / due date / checklist if present.
4. Confirm you did not create duplicate spam cards on purpose.
