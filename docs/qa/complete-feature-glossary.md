## 📖 Chapter 6: The Definitive GRC Operational Glossary

To pass your final independent online technical evaluation, you must master the exact plain-English meanings of our core operational terminology:

* **ALE (Asset Loss Expectancy):** The exact amount of money a company calculates it would lose annually due to unpatched cybersecurity risks or technical compliance failures.
* **BigInt Integer Cents Format:** Storing financial numbers in database rows as raw whole integers representing cents ($1.00 = 100$) instead of decimals, completely blocking floating-point math rounding errors during heavy multi-million dollar corporate audits.
* **Blast Radius Mapping:** The automated technical mapping process used to calculate exactly how many client accounts, server nodes, or interconnected databases will be corrupted or damaged if a single piece of software fails.
* **CSRD (Corporate Sustainability Reporting Directive):** A strict European law mandating that all large businesses provide verifiable data mapping out their physical environmental footprint, carbon intensity, and hardware efficiency factors.
* **Multi-Tenant Memory Bleed:** A catastrophic security failure where private data belonging to one company slips past code perimeters and displays on the screen of an entirely different company.
* **RLS (Row-Level Security):** A hardware database-level protection layer that validates a user's security badge before allowing individual database rows to display, ensuring complete client isolation.
* **Skeleton Loading Frame:** Temporary gray placeholder blocks that flash on screen during a tenant swap, proving the system is safely purging active cache memory to prevent cross-tenant data leaks.
* **TAS (Tenant Architecture Specifications):** The unchangeable code constitution that governs how the application manages data, security, and agent boundaries.
* **Zero-Trust Ingestion:** A strict security architecture policy that treats all incoming data as potentially malicious, requiring mandatory filtering through a sanitization DMZ gateway (**`Irongate`**) before it ever reaches the server databases.
