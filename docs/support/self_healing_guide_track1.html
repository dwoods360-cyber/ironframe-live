What happened?

Ironwatch (Agent 15) compares live configuration hashes against the constitutional baseline in docs/TAS.md. When you change a rule without an approved amendment, the UI freezes and shows GOVERNANCE DRIFT DETECTED.

Recovery steps (lab)

  1. Click Simulate drift below to trigger the freeze.
  2. Click the amber FREEZE COMMAND POST button (same label as production top sub-header).
  3. Wait 3 seconds while Irontech (Agent 4) runs the self-healing loop.
  4. Listen for the resolution chime — baseline restored.
