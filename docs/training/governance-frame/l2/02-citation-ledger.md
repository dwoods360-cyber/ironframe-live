## Lab

Build a ledger for your draft:

| Claim (short) | Source (title + publisher) | Primary? | Supports claim exactly? | Notes |
|---------------|----------------------------|----------|-------------------------|-------|
| | | Y/N | Y/N/Partial | |

Add draft ending:

```markdown
### V. Sources & Citations
1. …
```

Never invent citations. If AI suggested a source, open it or mark Unsupported.
