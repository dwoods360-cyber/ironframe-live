# Ops / GTM glossary (internal operators)

**Track:** OPS_GTM · **Reading level:** 11th–12th grade  
**Not for:** Classroom Core students or partner tenants

| Term | Plain meaning |
|------|----------------|
| **Ops Hub** | Internal control surface at `/dashboard/operations`. |
| **SUSPECT** | Early lead from Ironleads — not ready for a design-partner ask. |
| **PROSPECT** | Qualified enough for outreach drafts. |
| **Approvals** | HITL queue before any send. |
| **DISPATCH** | Human-approved send (email or SMS). |
| **PURGE** | Retire a bad draft without sending. |
| **Design-partner seat** | Paid commercial offer (~$4,999 / ~90 days). **Not** training Path B (Platform Practitioner). |
| **LIVE assist** | Mic sidecar on workflow-review — you still host the call. |
| **Operator library** | GTM playbook directory in Ops Hub. |

Full operator glossary: `docs/sales/design-partner-gtm-operator-glossary.md`
