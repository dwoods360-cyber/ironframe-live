# Level 2 Training Index — Practitioner Track

**Audience:** Practitioners and trainers. **Release:** `v0.1.0-ga-epic17`.

Finish Level 1 first if the Command Deck is new to you. Then work these chapters in order.

Keep a journal. Write the route. Write what you saw. Write the time.

---

## Before you start

1. Get lab access to the app host.
2. Get IronBoard access when a chapter needs the bridge.
3. Use trainer logins for admin-only routes. That includes `/admin/onboarding`, `/dashboard/admin/approvals`, and `/audit`.
4. Keep labs short. Do the steps. Do not paste long dumps.
