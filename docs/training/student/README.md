# Student section — Ironframe Core (SaaS)

**Audience:** High-school classroom students and internal trainers  
**Reading level:** 11th grade  
**Scope:** Operating the **Ironframe Core** SaaS app (tenant cockpit), not the ops/GTM stack

This folder is the home for every handbook that teaches students how to use Ironframe Core. If a page helps that learning, it belongs here at 11th-grade reading level.

## Out of scope for this track

Do **not** use these for student Core labs:

- Ops Hub (`/dashboard/operations`) and Operator library  
- Ironleads / SalesTeam / Approvals DISPATCH / design-partner sales motion  
- Workflow-review LIVE assist and Teams Graph GTM tools  
- Design-partner GTM playbooks under `docs/sales/`  
- Level 2 technical / deployment manuals  

Partners use [LEVEL1-PARTNER-INDEX](../LEVEL1-PARTNER-INDEX.md) and `docs/user-manuals/`. Instructors who need ops/GTM use Ops Hub docs separately.
