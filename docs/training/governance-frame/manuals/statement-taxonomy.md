# Statement taxonomy (GF Reader / Writer)

**Authority:** GF-STANDARDS-001 §5  
**Reading level:** 11th grade

Every important sentence in a Governance Frame publication should fit **one** of these:

| # | Type | Meaning | Label tip |
|---|------|---------|-----------|
| 1 | **Documented fact** | Directly supported by a checked source | “According to [source]…” |
| 2 | **Analytical interpretation** | GF’s reasoned conclusion from facts | “This suggests…” |
| 3 | **Illustrative example** | Hypothetical or simplified scenario | “For example (illustrative)…” |
| 4 | **Architectural recommendation** | Possible control / governance response | “Organizations may…” |
| 5 | **Product relevance** | How a named product *might* implement part of #4 | Rare; disclose clearly |
| 6 | **Unresolved research question** | Not yet answered by enough evidence | “Open question…” |

**Rule:** Do not blend types in one sentence (for example, do not state an interpretation as if it were enacted law).

### Reader drill

Open any published briefing. Highlight three sentences and assign each a type number (1–6). If you cannot decide, the prose is probably blended — note that as a finding.
