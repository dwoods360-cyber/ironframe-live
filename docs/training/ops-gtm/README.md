# Internal Ops / GTM track (elective)

**Audience:** Internal Ironframe operators (GLOBAL_ADMIN / BUSINESS_ADMIN)  
**Reading level:** 11th–12th grade plain English  
**Prerequisite:** Trainer assignment — **not** part of Jr. GRC Analyst graduation

This track covers Ops Hub, Approvals HITL, Ironleads, and workflow-review LIVE assist.  
It is **not** required for GRC analysis careers.

Student Core and Analyst tracks stay on Ironframe Core SaaS only.

**Briefings / promote:** Do **not** treat Ops Hub Approve as a substitute for [Governance Frame Path D](../governance-frame/README.md). Learners should hold **GFP-Writer** (ideally **GFP-Verifier**) before practicing production promote judgment. Queue runbook: `docs/briefing-queue/README.md`.
