# Marketing Plan — Ironframe GRC

## Positioning

**Category:** Quantitative GRC / sovereign risk platform (see [StoryBrand Framework](./storybrand-framework.md) and [Marketing Strategy Library](./marketing-strategy-library.md))  
**Tagline:** Quantified risk. Zero-trust ingestion. One Command Center.  
**Proof:** GA v0.1.0-epic17, cloud integration suite, TAS constitutional spec

## Target audiences

1. **CISO / VP Security** — financial risk narrative, Ironwatch, quarantine
2. **GRC directors** — Irontally, exports, audit intelligence
3. **CFO / risk** — ALE, insurance savings, governance dividend
4. **ESG leads** — carbon pulse, physical units, resilience streak
5. **Technical evaluators** — API docs, integration smoke, open architecture story

## Channels

| Channel | Tactic | Framework alignment | KPI |
|---------|--------|---------------------|-----|
| **Website / product** | Role-based landing pages; StoryBrand wireframe | StoryBrand, Obviously Awesome | Demo requests |
| **LinkedIn** | Agent spotlight series, GA evidence posts | Made to Stick, Contagious | Engagement, inbound |
| **Technical blog** | Epic deep-dives (WORM, tenant isolation, LKG) | They Ask You Answer | Developer signups |
| **Webinars** | Command Center live demo | Challenger Sale | Pipeline SQL |
| **Partners** | MSSP Command Center story | Crossing the Chasm, whole product | Partner-sourced tenants |
| **Events** | RSA / Gartner-style talk: "Beyond the heatmap" | Play Bigger category POV | Meetings booked |

## Campaign themes (2026 H1)

1. **GA launch** — Release evidence, integration proof, design-partner quotes (labeled)
2. **Sustainability truth** — Ironbloom physical units vs greenwashing narratives
3. **Audit-ready** — Ironquery exports + WORM roadmap honesty
4. **Tenant sovereignty** — Multi-tenant without bleed

## Content pillars

- **Educate:** ALE explainer, zero-trust ingest primer (They Ask You Answer)
- **Prove:** Release evidence packs, test suite results (authority / social proof)
- **Humanize:** Agent roster personas (Ironwatch, Ironlock, etc.) — customer as hero
- **Convert:** Demo CTA, pilot program (clear single CTA per asset)

## Compliance in marketing

- No guaranteed compliance certification claims unless contracted
- Simulation/shadow features labeled "staging" in public copy
- Financial baselines (Medshield $11.1M, etc.) labeled "illustrative demo tenants"
- Competitive statements: Shipped / Pilot / Roadmap only
